# my_1stproject
Developed a Temperature Converter 
to convert values between Celsius, 
Fahrenheit, and Kelvin using core programming logic.

📌 🔹 Project Briefing

Temperature Converter is a simple yet effective application that allows users to convert temperature values between different units such as Celsius, Fahrenheit, and Kelvin. The project is designed to demonstrate fundamental programming concepts including user input handling, conditional statements, and mathematical calculations.

The application takes a temperature value as input, processes it based on the selected unit, and provides accurate conversions into other temperature scales. It is a beginner-friendly project that highlights problem-solving skills and basic logic building.

📌 🔹 Project Summary

The Temperature Converter project focuses on building a basic utility tool that performs temperature conversions efficiently. It ensures accuracy and simplicity while helping users understand how different temperature units relate to each other.

Through this project, core programming concepts such as decision-making, formula implementation, and input/output operations are applied in a practical way. This project serves as a strong foundation for developing more advanced applications with user interfaces and additional features in the future.
